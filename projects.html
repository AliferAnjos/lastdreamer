1- Mouse Malandro:

Malandro means Rascal/Rogue on portuguese.
Mouse Malandro will move your mouse cursor every 5 seconds.
This will allow you to stay AFK (Away From Keyboard) without getting your screen blocked.
Perfect for meetings and classes you don't want to attend, but need to keep your notebook on.PS: But you didn't hear this from me ;)

Instructions:

Start button to start the moving of your mouse cursor. The button will change color on "Running Mode".
Stop button to stop de movement, one click is enough, but it won't be displayed a button pressed movement.
In case of crash, finish the program on your task manager!
The Linux Version might run on macOS, but was not tested yet.

SINCE IS A FILE .EXE, You'll have to allow its use, but it's completely HARMLESS.

Last Dreamer - Network Checker

This program will collect network data run ping tests to check your network connection.
As soon as the tests are finished, you can save the report on a .txt file and send it to your Network Analyst
All the information you'll need to troubleshoot on a basic level on a simple click of the mouse!
Not all of us know how to extract the kind of information, so it's a way to make it easier for all.

Instructions:

Allow the use on your machine and simply click on "Diagnose" Button.
On the Windows version, it will take approximately 2:35 minutes to complete the tests.
After saving the Report file, the program can be closed.
PS: My OS is on portuguese, but the data will be collected on your on language, or most of them!

Download Here: Last Dreamer - Network Checker - For Windows
